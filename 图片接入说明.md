# 图片接入说明

我已经把 `assets/image` 中的图片复制到 `assets/resources/image`，并让 `AutoDemoGame.ts` 在运行时自动加载这些图片。

当前自动匹配关系：

- `茶铺.png`：主背景
- `桌椅.png`：座位和操作台占位
- `书生.png`：客人形象
- `对话框.png`：客人点单气泡
- `九宫格.png`：按钮底图
- `茉莉绿茶.png`：已预留为绿茶/茶饮素材
- `蜜香红茶.png`：已预留为红茶/茶饮素材

运行方法：

1. 在 Cocos Creator 中停止预览。
2. 如果资源没有刷新，右键 Assets 面板选择刷新，或点 Developer → Reload。
3. 打开 `assets/scene/Demo.scene`。
4. 点击运行。

如果 Console 出现“图片加载失败”，请确认对应图片存在于：

`assets/resources/image/图片名.png`

注意：Cocos 的 `resources.load()` 只能直接加载 `assets/resources` 目录里的资源，所以我保留了你原来的 `assets/image`，同时复制了一份到 `assets/resources/image` 供代码运行时加载。
